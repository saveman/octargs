Head utility (head)

Basic version of 'head' shell utility.

  • Manual parsing of argument values

Cat tool (cat)

Basic version of 'cat' shell utility.

Features shown in the example:

  • Exclusive, valued, switch, positional arguments.
  • Storing parsed values in provided storage variable including type checking.
  • Usage (help automatic generation).

Calculator (calc)

Performs basic math operations on specified values.

Features shown in the example:

  • Exclusive, valued, switch, positional arguments.
  • Getting results with type (get_first_value_as()).
  • Usage (help automatic generation).

File get/download (getfile)

Calculates sum of given values.

Features shown in the example:

  • Custom dictionary usage.

Basic version of 'head' shell utility with Windows-like command line arguments parsing.

Features shown in the example:
