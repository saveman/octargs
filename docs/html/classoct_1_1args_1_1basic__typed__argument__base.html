Detailed Description

template<template< typename, typename, typename > class derived_TT, typename char_T, typename values_storage_T, typename data_T>
class oct::args::basic_typed_argument_base< derived_TT, char_T, values_storage_T, data_T >

Argument base type handling.

The class allows to set following "hooks" (functions):

  • convert - used to convert the string representation to data type
  • check - used to check if the data after conversion is valid
  • store - used to store the data
