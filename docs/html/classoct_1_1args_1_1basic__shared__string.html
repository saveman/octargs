Detailed Description

template<typename char_T>
class oct::args::basic_shared_string< char_T >

String wrapper using shared pointer.

This class wraps std::basic_string stored using shared pointer. It is used to store strings safely in exceptions as shared pointer copy construction does not throw exceptions.
